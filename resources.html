Resources

Want to contribute content but not sure where to start? Or just need some help getting a project off the ground? We’ve taken a stab at answering these and other questions below.

How do I add a dataset or project?

A: First you have to become a contributor! To do that, you’ll need to follow these steps.

1. Create a Github account
2. Open the homepage and click login at the top right.
3. Sign in with your github credentials.
4. Authorize access in github.
5. Return to portal, refresh the page, and begin.
